# Continuous Integration and Continuous Delivery (CI/CD) with GitHub Workflows

This repository contains the material used for the CI/CD workshop at the HSF/IRIS-HEP Analysis Reproducibility Training event. The material is based on the CI/CD portion of the [Software Engineering for Scientific Computing Course](https://henryiii.github.io/se-for-sci/content/intro.html) and the [CI/CD Course](https://hsf-training.github.io/hsf-training-cicd-github/) by the [HEP Software Foundation](https://hepsoftwarefoundation.org/).

The live slides can be found [here](https://ariostas-talks.github.io/2026-04-29-cicd).
